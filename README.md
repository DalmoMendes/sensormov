# SensorMov

Reconstrução do App Sensor de Movimento criado no Bootcamp Decola Tec 3ª edição na DIO com React Native.

## Recursos usados 

- Libs react-native-shake e react-native-torch
- StyleSheet
- Hook useState
- Hook useEffect
- Add Listener to RNShake
- Lifecicly ReactJS
